# Muddy Sunset Worker

Starter limpo para evoluir o projeto com:

- Hono (HTTP router)
- Drizzle ORM (D1/SQLite)
- TypeScript
- Zod (validações)
- Discord bot por Interactions
- Bun como package manager/runtime de desenvolvimento

## Primeiros passos

```bash
bun install
bun run dev
```

## Banco local (D1)

```bash
bun run db:migrate:local
```

## Deploy

```bash
bun run deploy
```
